Ψυχολόγος

Σχετικά

Ο χώρος που δεν χρειάζεται να «αποδείξετε» τίποτα.

Είστε ήδη αρκετοί. Εγώ είμαι εδώ για να βάλουμε μαζί μια τάξη στο μέσα σας, με τρόπο πρακτικό και ήπιο.

Για ποιους ανθρώπους ταιριάζει

Για όσους νιώθουν άγχος, πίεση, ενοχές, θυμό ή κούραση. Για όσους περνούν αλλαγές, απώλειες, δυσκολίες στις σχέσεις, ή απλώς θέλουν να καταλάβουν καλύτερα τον εαυτό τους.

Τι να περιμένετε

Θα ακούσω πρώτα, χωρίς να σας «χωρέσω» σε ταμπέλες. Μετά θα βρούμε μαζί τι είναι αυτό που πονάει, τι το κρατάει ζωντανό, και ποιο είναι το πιο απλό επόμενο βήμα.

Υπηρεσίες

Όχι «πακέτα». Ένα μονοπάτι που ταιριάζει σε εσάς.

Το σημαντικό είναι να ξεκινήσουμε από αυτό που σας βαραίνει, και να φτιάξουμε μια διαδρομή που να βγάζει νόημα.

Ατομικές συνεδρίες

Για άγχος, πίεση, σχέσεις, αυτοεκτίμηση, όρια και δύσκολες αποφάσεις.

Συμβουλευτική γονέων

Πρακτική καθοδήγηση για τη συμπεριφορά, τα όρια, και τη σχέση με το παιδί.

Παιγνιοθεραπεία

Ένας ασφαλής τρόπος ώστε το παιδί να εκφράσει αυτό που δεν μπορεί να πει με λόγια.

ABA υποστήριξη

Δομημένη προσέγγιση με στόχο δεξιότητες, λειτουργικότητα και καθημερινή αυτονομία.

Δεν χρειάζεται να ξέρετε «τι ακριβώς θέλετε» για να επικοινωνήσετε.

Αρκεί να μου πείτε τι σας δυσκολεύει αυτή την περίοδο. Θα σας βοηθήσω να βρούμε το σωστό ξεκίνημα.

Πώς δουλεύουμε

Μικρά βήματα. Καθαρά. Χωρίς πίεση.

Θα ξέρετε πάντα τι κάνουμε και γιατί. Και θα έχετε χώρο να πείτε «όχι ακόμη» όταν το χρειάζεστε.

1. Πρώτη επικοινωνία

Μου γράφετε τι σας απασχολεί. Σας απαντώ με ηρεμία και σαφήνεια.

2. Συμφωνούμε τον στόχο

Δίνουμε όνομα σε αυτό που πονάει και βρίσκουμε τι θα σας ανακουφίσει.

3. Ρυθμός που αντέχετε

Χτίζουμε σταθερότητα, δεξιότητες και στήριξη, βήμα βήμα.

Αν μέσα σας υπάρχει ένα «ίσως», είναι αρκετό για να ξεκινήσουμε.

Στείλτε ένα μικρό μήνυμα με δύο προτάσεις. Θα σας απαντήσω με το πιο ήπιο, καθαρό επόμενο βήμα, χωρίς πίεση.
